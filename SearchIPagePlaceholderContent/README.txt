Do yo ask yourself questions like these:
 - Which pages contains a table?
 - Which pages contains the class "read-more"?
 - Which pages contains style attributes?
Use this tool to search placeholder content HTML.

C1 4.0 required

Copy SearchIPagePlaceholderContent.bat to the project root (beside init.bat) and run it.
Give yourself access to the new perspective called Developer Tools (press F5).
Click on "Search placeholder content".
